# 2. Justificación del Proyecto

Actualmente en la mayoría de empresas no existe un sistema automático de organización de proyectos que pueda centralizar la información de tareas ni medir la productividad en tiempo real. Esto limita la mejora de los retrasos, el desbalance en la carga de trabajo y la poca visibilidad del panorama del desarrollo para el manager. En vista de esta situación, durante las reuniones del equipo de desarrollo de software de Oracle se ha determinado implementar un sistema que mejore la productividad y la visibilidad de las actividades de cada miembro del equipo.

---

# 3. Objetivos del Proyecto

Disminuir el tiempo promedio de desarrollo de proyectos en un 20%.
Disminuir la cantidad de tareas incompletas en un 30%.
Implementar la interacción con el chatbot de Telegram que alcance una calificación mínima de 4/5 en usabilidad, que se puede evaluar mediante encuestas.


---

# 4. Alcance de Alto Nivel

## Incluye:

- Prototipo de Portal Web: Entorno en línea con el estado (actividades, tareas, sprints) de proyectos en tiempo real en forma de panel de métricas con un chatbot impulsado por IA con interacción personalizada al usuario.
- Funcionalidad para que los Líderes y managers consulten, agreguen y eliminen tareas.
- Integración con la infraestructura y procesos de Oracle Cloud.
- Funcionalidad para que los desarrolladores visualicen las tareas que les son asignadas y hagan uso del chatbot para visualizar métricas personales.

## No incluye:

- Aplicación móvil o de escritorio.
- Soporte técnico a futuro.
- Desarrollo de un LLM desde cero.
